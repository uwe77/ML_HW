# Discussion
## 1.請問在grid search的結果中,C的大小與分類率高低有何關係?
從圖表中我們觀察到，在 sigma=(-155 ~ -40) 和 (110 ~ 150)^1.05 的範圍內，對交叉驗證（CV）的影響似乎並不明顯。\
在(-155 ~ -40) 的範圍內，可能是因為數據在核空間中的分佈過於雜亂，導致超平面難以區分兩組數據，因此對 C 的影響微乎其微。\
然而，在(-40 ~ 100) 的範圍內，我們可以明顯地看到 C 對 CV 的影響急劇增加，而且在這個區間內的 CV 高達90%以上。

## 2.sigma的大小的改變與分類率是否有關係? 若有,請探討sigma的差異與特徵的數值有什麼關聯性?
有著密切關西。當 sigma 過低時，數據在核空間中的分佈並不理想，兩個類別混在一起，導致超平面無法有效區分兩個類別。因此，交叉驗證（CV）的準確率甚至可能低於50%。 \
在具有RBF核的SVM情境中，sigma參數決定了徑向基函數的寬度。徑向基函數是特徵空間中數據點相似性的度量。\
較小的sigma意味著附近點的影響更大，使得決策邊界對數據的局部變化更敏感。相反，較大的sigma會使決策邊界更平滑，減少對局部波動的敏感性。

## 3.若分析過程不採用two-fold cross validation，則分類率是否會更高? 請探討之。
有時確實可能達到更高的準確率，從非交叉驗證的圖表中也可以觀察到，某些情況下的交叉驗證分數比2-fold更高。然而，這可能只是因為測試數據在相應的訓練模型下表現較好的偶然情況。
